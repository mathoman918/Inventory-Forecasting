# Inventory Forecasting & Demand Analysis

## Project Overview
This project analyzes retail inventory and demand forecasting data to identify sales trends, forecast accuracy, stockout risks, and the impact of promotions and seasonality on sales performance.

The goal of the project is to improve inventory planning and demand forecasting to reduce stockouts and optimize inventory levels.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Excel (Pivot Tables & Charts)
- SQL (Data Processing)
- Power BI / Dashboard
- GitHub

## Project Steps
1. Data Cleaning and Preparation
2. Exploratory Data Analysis
3. Forecast Accuracy Analysis
4. Inventory & Stockout Analysis
5. Promotion & Seasonality Analysis
6. Dashboard and Presentation

## Key Insights
- Promotions significantly increase sales.
- Winter season shows the highest average sales.
- Forecast accuracy is around 84% (MAPE ~16%).
- Approximately 15% of inventory records show stockout risk.
- Clothing category has the highest demand.

## Conclusion
Improving demand forecasting and adjusting inventory levels during promotion periods and high-demand seasons can significantly reduce stockouts and improve revenue.
